# PC Pressure Monitor 1.0.1

Reliability and storage maintenance release.

## Highlights

- dedicated lightweight uninstaller;
- deterministic removal order: service/driver, application, snapshot logs, self-cleanup;
- compact historical snapshots every 30 seconds (roughly 8 MB/day under test load);
- bounded in-memory history without duplicated command lines or TCP endpoint arrays;
- fast single-instance installer with Update/Repair mode;
- selectable installation directory;
- legacy log files separated from the new `snapshots-YYYY-MM-DD.jsonl` format.

Download the installer and portable package from [GitHub Release v1.0.1](https://github.com/Solpadoin/SpywareMonitor-PC-Pressure-Monitor-/releases/tag/v1.0.1).
